# FrontEnd

## Pacotes utilizados

- Vite (compilador e minificador de codigo frontend)
- Tailwind (framework para css)
- shadcn (Lib para componentes pre moldados para o frontend)

## Como rodar o projeto 

### DOCKER
   ```bash
   docker-compose up --build
  ```

### NPM
```bash
npm install
npm run dev
```
